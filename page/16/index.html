More than 3 levels of stable

Fri Oct 24, 2008 by Caleb Cushing in  Regen2, pms

Currently Gentoo has arch, ~arch, and ~M ( - if not available ). One of the main things people want are security patches only updates. I tend to agree with this, although they should be between arch and ~arch, maybe +arch. Technically a security patch can break things which is why they shouldn’t be considered stable. A new tarball should be released, 1 week after a new glibc or gcc is updated, for sure, but any system package should be enough.
